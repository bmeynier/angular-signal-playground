Comment lire ce schéma

  • HeaderComponent → Affiche le titre en temps réel et l’horloge via ClockService.
  • DashboardComponent → Formulaire, liste filtrée, graphique Chart.js, simulation de tâches.
  • DashboardService → Contient toutes les données et la logique métier :
    • tasks = signal qui contient la liste des tâches
    • filteredTasks = computed qui filtre les tâches selon filter
    • title = signal du titre affiché dans le header
    • effects = log des tâches, sauvegarde localStorage, mise à jour du titre, envoi de stats
  • ClockService → Signal now mis à jour chaque seconde pour l’horloge

Astuce pour se souvenir

  • Signal = boîte magique qui prévient Angular quand ça change
  • Computed = règle magique recalculée automatiquement
  • Effect = observateur automatique qui réagit aux changements
  • Template = LEGO : Angular assemble tout automatiquement
  • OnPush = Angular ne regarde que les boîtes qui changent
